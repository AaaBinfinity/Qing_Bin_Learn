# Git的安装和初始化配置

## 一、安装

1. 去[git官网](https://git-scm.com/)进行下载并安装
2. 验证安装：在终端输入`git -v`命令



看到版本信息说明安装成功

------

## 二、Git的使用方式

### 1.命令行

最基本、最常用，在终端中使用git命令来操作Git

Git的所有命令都以`git`开头

### 2.图形化界面（GUI）

通过一些专用的图形化工具使用Git

### 3.IDE插件/扩展

在IDE中通过扩展或者插件来使用Git（VSCode默认集成了源码管理器）
